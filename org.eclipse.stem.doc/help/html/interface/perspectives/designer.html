Designer Perspective

The Designer perspective is loaded the first time that you load STEM with a set of default Views. Subsequent uses of STEM will load the perspectives and views that you last used.

The purpose of this perspective is to hold the windows that are used in building a Scenario that you will then run. You can also use it to start an existing simulation.

To start an existing simulation, select the Scenario View. It will show an existing projects and if you select a project, it will display the Scenario for that project. Double click the Scenario to select it and start the simulation running. STEM will automatically switch to the Simulation Perspective

The following views are displayed for the Designer Perspective by default. However you can close any of these views and open others.

Default Views

  • STEM Project Explorer
    • This view is used to build new STEM projects.
  • Tabbed viewsThe tabbed view holds the following views.
    • Scenario View
      • When selected this view will display a list of existing projects with their Scenarios .
    • Models View
    • Graphs View
    • Sequencer View
    • Decorators View
    • Properties View
    • Tasks View
    • Error Log View
