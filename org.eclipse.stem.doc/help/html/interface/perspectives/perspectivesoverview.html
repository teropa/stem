Perspectives

STEM is an Eclipse Rich Client application. As such it is organized into "perspectives" each of which contains one or more views.
The contents of a perspective can be customized by the user but the default perspectives are designed to support different uses of the STEM application.

The STEM system contains three "perspectives":

  • The Analysis perspective is use to analyze epidemic data  gathered from completed simulations or from import public health reports. It also supports comparison and validation of multiple data sets.
  • The Designer perspective is used to design and create new STEM Simulations.
  • The Simulation perspective is used to start,  view, and control running STEM Simulations.
