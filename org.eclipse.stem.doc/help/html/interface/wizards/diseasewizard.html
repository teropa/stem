The New Disease Wizard is used to create and define the properties of a new Disease. A file name for the new instance along with the Disease Name must be provided in order to be able to click "Next" or "Finish".

Parameters that can be specified depends upon the particular disease model selected in the Disease Model drop-down. For example, for an SI model, the parameters are:

  • Disease Name
  • Population: i.e., human
  • Time Period (TP): In milliseconds
  • Frequency Dependent: Yes or No
  • Reference Pop Density: PM/SQ KM
  • Transmission Rate (Β)
  • Non-Linearity Coefficient: Must be greater than or equal to 0.0
  • Infectious Recovery Rate (γ): PM/TP*
  • Infectious Mortality Rate (μ ι): PM/TP*
  • Characteristic Mixing Distance: In kilometers
  • Road.Net.Inf.Proportion: Fraction per Road
