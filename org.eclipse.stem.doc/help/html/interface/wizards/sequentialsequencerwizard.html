The New Sequential Sequencer Wizard is used to create and define the properties of a new Sequential Sequencer. A name must be provided for the Sequential Sequencer in order to be able to click "Next" or "Finish". The length of the Cycle Period can be specified, and the unit of time (milliseconds, seconds, minutes, hours, days). The Start Date can be chosen, as well as the having the option of specifying an End Date.
