The New Population Model Wizard is used to define the properties of a new population model. A file name must be provided for the Population model.

Properties that can be manipulated within the New Population Model Wizard depends on the model selected from the drop-down. For a standard population model they are:

  • Population: i.e., human
  • Model Name
  • Background Birth Rate: PM/TP*
  • Background Mortality Rate (μ): PM/TP*
  • Time Period (TP): In milliseconds
  • Location ISO Key: If no location is specified, the population model is assumed global
