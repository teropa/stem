The New Population Initializer Wizard is used to define the initial state of a population. A name must be provided for the Population Initializer in order to be able to click "Next" or "Finish".

Properties that can be manipulated within the New Population Initializer Wizard depends on the Population Initializer selected from the drop-down. For a standard population initializer, they are:

  • Individuals: Count or Count/Square Kilometers
  • Use Density: Initialize using density instead of an absolute number
  • Population: i.e., human
  • Location ISO Key
