The New Graph Wizard is used to create and define the properties of a new Graph. A name for the Graph must be provided.

Properties that can be specified depends upon the source selected in the drop-down. For a square lattice graph, the properties are:

  • Source: Drop-down menu selection
  • X Size (Units)
  • Y Size (Units)
  • Nearest Neighbors: Yes or No
  • Next Nearest Neighbors: Yes or No
  • Periodic Boundaries: Yes or No
