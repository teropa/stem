Views

The STEM interface includes a number of different "Views". Each view displays a particular aspect of the state of STEM, its components, or of an active Simulation. Some views have functions that allow Simulations to be created and controlled.

In addition, views that list Simulation components (Decorators, Graphs, Models, Scenarios and Sequencers) also support an instant "pop-up" display of the Dublin Core metadata of the component displayed in the view. See Dublin Core Pop-up for more details.
