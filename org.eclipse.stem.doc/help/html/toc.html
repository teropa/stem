STEM Table of Contents

The Spatiotemporal Epidemiological Modeler (STEM) is an extensible software system and framework for developing and running discrete event simulation models. It was primarily developed for modeling the spread of infectious diseases, but it is general enough to be applied to virtually any other modeling application involving space and time.
